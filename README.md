## Descripcion

API REST de catalogo de productos construida con Spring Boot 3.2.x, Java 17,
Maven, Spring Web, Spring Data JPA, Validation y H2. El proyecto separa las
responsabilidades por capas y usa DTOs para controlar los datos de entrada y
salida de la API.

La documentacion completa de ejecucion, arquitectura y evidencias se termina al
cerrar los checkpoints del Post-Contenido 1.
